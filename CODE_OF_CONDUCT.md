# Code of Conduct

We want fluxtty to be a project where people can collaborate productively and respectfully.

## Our standards

Examples of behavior that help build a positive community:

- being respectful and patient
- welcoming contributors with different levels of experience
- giving actionable, technical feedback instead of personal criticism
- assuming good intent while still addressing problems directly
- respecting privacy, security, and the time of other contributors

## Unacceptable behavior

Examples of unacceptable behavior include:

- harassment, discrimination, or intimidation
- insulting or demeaning comments
- trolling, bad-faith arguing, or intentionally derailing discussion
- publishing private information without permission
- repeated hostile behavior after being asked to stop

## Scope

This code of conduct applies in issues, pull requests, discussions, code review, and other project-managed community spaces.

## Enforcement

Project maintainers may remove comments, close conversations, reject contributions, or take other appropriate moderation action when behavior violates this code of conduct.

If you need to report a conduct issue, contact the maintainers privately through GitHub or another private contact method published by the project. Please do not open a public issue for conduct reports.
